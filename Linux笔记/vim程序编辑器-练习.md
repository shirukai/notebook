# vim程序编辑器-练习

### 1. 在 /tmp 这个目录下建立一个名为 vitest 目录；

```
mkdir -p /tmp/vitest
//-p意思是如果父目录不存在，自动创建

```

### 2. 进入 vitest 目录；

```
cd /tmp/vitest
```

### 3. 将 man.config 复制到本目录底下；
用文件传输工具
### 4. 使用 vi 开启本目录下的 man.config 这个文件；

```
vi man.config
```

### 5. 在 vi 中设定一下行号；

```
：set nu
```
